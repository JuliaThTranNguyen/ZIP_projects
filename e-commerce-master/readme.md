## This project will be deployed on Heroku 
#### What is Heroku?
Heroku is a cloud platform as a service supporting several programming languages.

One of the first cloud platforms, Heroku has been in development since June 2007, when it supported only the Ruby programming language, but now supports Java, Node.js, Scala, Clojure, Python, PHP, and Go.

#### Documentations
[Link](https://devcenter.heroku.com/)

#### Method of deployment
##### Heroku CLI commands
```
cd your-springboot-app
git init 
heroku create pick-any-name-for-your-app
git add .
git commit -m "deploy to heroku"
git remote add heroku https://git.heroku.com/pick-any-name-for-your-app.git
git push heroku master
```
